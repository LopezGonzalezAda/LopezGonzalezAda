### 📝 Azota Exam Auto-Grading Pipeline

An end-to-end computer vision system developed with a four-person team in collaboration with Azota, a Vietnamese EdTech company.

I worked primarily on the AI and computer-vision pipeline:

* OpenCV document standardisation for angled and low-quality phone photos;
* YOLOv8 layout detection across multiple exam-sheet templates;
* a custom Optical Mark Recognition engine using Hough Circle Transform;
* detection post-processing and ambiguity-resolution logic;
* OCR-assisted question-number and essay-score extraction;
* automated grading and visual feedback.

The final YOLOv8l model achieved **0.995 mAP50** and **0.858 mAP50–95** across nine exam-region classes.

**Technologies:** Python, PyTorch, YOLOv8, OpenCV, EasyOCR, PaddleOCR, Streamlit, SQLite

[View project](https://github.com/ChadiaWillems/Industry_Project_Azota)

### 🎹 AI-Powered Piano Trainer

A real-time computer vision and embedded-systems project that detects pressed piano keys and provides immediate feedback to the user.

I collected and labelled a custom dataset, trained a YOLOv8 model and built a Raspberry Pi-to-laptop pipeline using socket communication for video streaming, inference and feedback.

The physical system includes a webcam, Raspberry Pi, LCD display, buzzer and CSV-based progress logging.

**Technologies:** Python, YOLOv8, OpenCV, Raspberry Pi, Socket Communication
